# Steps to run the app

1. npm install
2. npm start (Scan QR code to start the expo app on your device, or press "i" (ios) or "a" (android) to start on a emulator)

## OR

2. npx expo run:ios/andriod (Run a local build on emulator or a physical device)

Reference: https://docs.expo.dev/get-started/set-up-your-environment/
